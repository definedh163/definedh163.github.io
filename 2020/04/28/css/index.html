flex 布局

本文部分资料引至阮一峰网络日志,仅限于个人使用.

如果你想知道更多,也许可以参考?阮一峰

常用属性

  • 指定弹性盒子

    display 指定元素为弹性容器(可以是行内块)

    属性值 flex;
       

   .box{

      display: flex;
   }

   @ : 设为 Flex 布局以后，子元素的float、clear和vertical-align属性将失效。

主轴对齐

  • 主轴对齐

    justify-content 主轴(默认横向)对齐方式

    属性值 flex-start | flex-end | center | space-between | space-around ;

    引用阮一峰的网络日志

      

       .box{

          display: flex;
          justify-content:flex-start;     @ : (默认值)沿着主轴左侧对齐
          justify-content:flex-end;       @ : 沿着主轴右侧对齐
          justify-content:center;         @ : 沿着主轴中间对齐
          justify-content:space-between;  @ : 贴合左右两端对齐, 子元素间隔相同
          justify-content:space-around;   @ : 贴合左右两端对齐,但是保留一定边框间隔, 子元素间隔相同大于边框间隔
       }

交叉轴对齐

  • 交叉轴对齐

    align-items 主轴(默认横向)对齐方式

    属性值 flex-start | flex-end | center | baseline | stretch ;

    引用阮一峰的网络日志

      

       .box{

          display: flex;
          align-items:flex-start;    @ : (默认值)沿着主轴左侧对齐
          align-items:flex-end;      @ : 沿着主轴右侧对齐
          align-items:center;        @ : 沿着主轴中间对齐
          align-items:baseline;      @ : 项目的第一行文字的基线对齐。
          align-items:stretch;       @ : 如果子元素未设置高度或设为auto，将占满整个容器的高度。
       }

改变主轴方向

  • 改变主轴方向

    flex-direction 改变主轴方向

    属性值 row | row-reverse | column | column-reverse ;

    引用阮一峰的网络日志

      

       .box{

          display: flex;
          flex-direction:row;              @ : (默认值)主轴为水平方向，起点在左端。
          flex-direction:row-reverse;      @ : 主轴为水平方向，起点在右端。
          flex-direction:column;           @ : 主轴为垂直方向，起点在顶部。
          flex-direction:column-reverse;   @ : 主轴为垂直方向，起点在底部。
       }

弹性元素折行

  • 弹性元素折行

    flex-wrap 如果弹性子元素宽度超过父级, 可以通过 flex-wrap 控制他们是否在父元素中折行

    属性值 nowrap | wrap | wrap-reverse ;

    引用阮一峰的网络日志 nowrap
    引用阮一峰的网络日志 wrap
    引用阮一峰的网络日志 wrap-reverse

      

       .box{

          display: flex;
          flex-wrap:nowrap;              @ : (默认值)不折行。
          flex-wrap:wrap;                @ : 折行, 第一行在顶部。
          flex-wrap:wrap-reverse;        @ : 折行, 第一行在底部。

       }

简写

  • 简写

    flex-flow 属性是 flex-direction 属性和 flex-wrap 属性的简写形式，默认值为 row nowrap。

    属性值 flex-direction || flex-wrap ;
      

       .box{

          display: flex;
          flex-flow:row nowrap;     @ : (默认值)主轴为水平方向,起点在左侧 不折行。
       }

    flex 属性是 flex-grow, flex-shrink 和 flex-basis 的简写，默认值为 0 1 auto。后两个属性可选。

    属性值 none | flex-grow flex-shrink? || flex-basis ;
      

       .box{

          display: flex;
          flex:row 1 1 auto;     @ : 放大1,缩小1,平均占比父容器

          该属性有两个快捷值：auto (1 1 auto) 和 none (0 0 auto)。

          建议优先使用这个属性，而不是单独写三个分离的属性，因为浏览器会推算相关值。
       }

放大比例

  • 放大比例

    flex-grow 为 0 时元素消失 空间被其他元素平均分配(值越大占比越大)。

    属性值 整数 ;

    引用阮一峰的网络日志

      

       .box{

          display: flex;

          .item1{
            flex-grow: 1; /* default 0 */
          }
          .item2{
            flex-grow: 2; /* default 0 */
          }

       }

缩小比例

  • 缩小比例

    flex-shrink 为 0 时根据元素的个数平均占比填充父容器空间(值越大占比越小)。

    属性值 整数 ;

    引用阮一峰的网络日志

      

       .box{

          display: flex;

          .item1{
            flex-shrink: 1; /* default 1 */
          }
          .item2{
            flex-shrink: 2; /* default 1 */
          }

       }

单独对齐

  • 单独对齐

    align-self 属性允许单个项目有与其他项目不一样的对齐方式，可覆盖 align-items 属性。默认值为 auto，表示继承父元素的 align-items 属性，如果没有父元素，则等同于 stretch。

    属性值 auto | flex-start | flex-end | center | baseline | stretch ;

    引用阮一峰的网络日志

      

       .box{

          display: flex;

          .item1{
            align-self: flex-start; /* default auto */     @ : 单独交叉轴顶部对齐
          }
          .item2{
            align-self: flex-end; /* default auto */       @ : 单独交叉轴底部对齐
          }
          .item2{
            align-self: center; /* default auto */         @ : 单独交叉轴居中
          }
          .item2{
            align-self: baseline; /* default auto */       @ : 单独交叉轴填充
          }
          .item2{
            align-self: stretch; /* default auto */        @ : 单独交叉轴基线对齐
          }

       }

显示顺序

  • 显示顺序

    order 属性定义项目的排列顺序。数值越小，排列越靠前，默认为 0。

    属性值 整数可以是负值 ;

    引用阮一峰的网络日志 wrap-reverse

    

         .box{
            display: flex;

            .item1{
              order: -1
            }
            .item2{
              order: 1
            }
         }

场景

场景一

  • 场景一

    需求: 做一个高度不写死,内容占容器 5/1 ,并且自动折行,自动居中 的卡贴选择器

  
div{
    display: flex;            @ : 指定弹性容器
    flex-wrap: wrap;          @ : 自动换行
    justify-content: center;  @ : 自动居中
    span{
        flex-basis: 20%;      @ : 设置子元素占用父元素宽度
    }

}

注意:设为 Flex 布局以后，子元素的float、clear和vertical-align属性将失效。
